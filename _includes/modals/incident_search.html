Sweet Search - Incident Based Twitter Search

This is a web-based search system used to retrieve incident specific information from social media. If a user searches about say, Paris attacks, we would aggregate information from twitter, show the various influencers, topics discussed and places of interest. The user would then be able to form a rounded opinion about the incident. Content tagging, faceted search and cross-document analytics were performed to achieve this. We were one of the few teams selected to present in class.
